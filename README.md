## Installation 
- Download the repo files
- Install necessary packages:

```
npm install
```
## Usage

- Start the server:
```
npm start
```

[Walkthrough of CRUD api routes here](https://drive.google.com/file/d/1F0dOgu8c5HU4F-nxipIhOEtBmlUaZH5g/view?usp=sharing)
